So what just happened? What does Javascript even do????

On the previous page, you used Javascript to "log" (write) the phrase "hello world!" in the console. The console is a place where the computer can say things for you to see.

Javascript is used to program the web. More recently, it has been used for other things like robots and apps.

Your browser has a hidden console. Its contents are shown below.

Why are there semicolons? Javascript doesn't usually require white space (spaces, tabs, or carriage returns), so it relies on semicolons to mark new lines. If you're using a modern browser, the semicolons aren't necessary as long as you press enter between lines.
